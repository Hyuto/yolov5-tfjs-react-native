## Run on Expo Go

**Setup**

Install main dependencies.

```bash
$ yarn
```

**Run the app~~**

```bash
$ yarn start
```

Open `Expo Go` app in your device and scan the QR code.

**Open WebApp**

```bash
$ yarn web
```
